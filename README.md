# React Native Workbook for Jetpack Compose Developers

## Who This Workbook Is For
This workbook is for Android developers who already build UI with Jetpack Compose and want to learn React Native with TypeScript in a practical way.

## Required Background
- Android development basics
- Jetpack Compose basics
- Kotlin basics

## What You Will Install
- Node.js (LTS)
- React Native CLI requirements (Java, Android SDK, emulator tooling)
- Android Studio
- Xcode only if you are on macOS and want to run iOS simulator

## Core Learning Method
You will create **one local React Native TypeScript project** and use that same project for all topics in this workbook.

Every topic file includes:
- Concept explanation
- Jetpack Compose comparison
- Working React Native TypeScript code
- Run instructions
- Practice task
- Practice area
- Screenshot proof section

## How to Use This Workbook in Sequence
1. Complete files in `00-course-setup/` first.
2. Move through folders from `01-...` to `10-...` in order.
3. In each topic:
   - Read the concept.
   - Compare with Compose.
   - Copy-paste the code into your local RN project.
   - Run the app.
   - Do the practice task.
   - Paste your practiced code into the topic file.
   - Add screenshot proof.
4. Use `98-practice-log/` to track your progress.

## Important Rule
This documentation workspace contains only markdown learning material. Build and run your React Native app in your own local project directory.
