# Create React Native TypeScript Project

## Goal
Create one local project that you will use for all workbook topics.

## Recommended Command
```bash
npx @react-native-community/cli init RNComposeWorkbook
```

## Verify Project
```bash
cd RNComposeWorkbook
ls
```
You should see files such as `App.tsx`, `package.json`, `android/`, `ios/`.

## Why One Project?
Using one project helps you focus on concepts instead of setup repetition.
Most beginner topics will ask you to replace full `App.tsx`.

## Keep This Rule
- Do not create the project in this documentation workspace.
- Create it in your own development location.
