# How to Use This Workbook

## 1. Read
Read one topic file at a time.

## 2. Compare With Compose
Map each React Native concept to the Jetpack Compose concept you already know.

## 3. Copy Code
Copy the provided TypeScript code exactly into your local RN project.

## 4. Run
Start Metro and run on Android/iOS.

## 5. Modify
Complete the practice tasks at the end of each topic.

## 6. Save Practiced Code
Paste your modified code into the topic's `Practice Area` section.

## 7. Add Screenshot
Add screenshot proof under the `Screenshot Proof` section.

## Daily Workflow Suggestion
- 1 to 2 topics per day
- Save notes in `What I Changed`
- Keep one project and evolve it topic by topic
