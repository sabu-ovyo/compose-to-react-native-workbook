# How to Use Practice and Screenshots

## Paste Practiced Code
At the bottom of every topic, paste your final practiced code in `My Practiced Code`.

## Add Notes
In `What I Changed`, write 2-3 short lines describing your modifications.

## Screenshot Storage Recommendation
Inside your local RN project, create:
```txt
screenshots/
```

## Screenshot Naming Style
```txt
01-01-react-native-vs-compose.png
02-01-usestate-counter.png
```

## Add Screenshot in Topic File
```md
![Screenshot](../screenshots/01-01-react-native-vs-compose.png)
```
